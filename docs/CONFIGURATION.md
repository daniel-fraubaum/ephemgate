# Configuration Reference

## Environment Variables / App Settings

### Self-Service Function App

| Setting | Default | Description |
|---------|---------|-------------|
| `STORAGE_ACCOUNT_NAME` | — | Name of the Azure Storage Account |
| `APPLICATIONINSIGHTS_CONNECTION_STRING` | — | App Insights connection string |
| `TAP_LIFETIME_MINUTES` | `60` | TAP validity duration in minutes |
| `TAP_IS_USABLE_ONCE` | `true` | Whether the TAP is single-use |
| `TAP_DISPLAY_TIMEOUT_SECONDS` | `300` | How long the TAP is visible in the frontend (seconds) |

### Helpdesk Function App

| Setting | Default | Description |
|---------|---------|-------------|
| `STORAGE_ACCOUNT_NAME` | — | Name of the Azure Storage Account |
| `APPLICATIONINSIGHTS_CONNECTION_STRING` | — | App Insights connection string |
| `TAP_LIFETIME_MINUTES` | `120` | TAP validity duration in minutes |
| `TAP_IS_USABLE_ONCE` | `true` | Whether the TAP is single-use |
| `TAP_DISPLAY_TIMEOUT_SECONDS` | `120` | How long the TAP is visible in the frontend (seconds) |
| `BLOCKED_GROUP_IDS` | — | Comma-separated list of Entra Group IDs to block from TAP issuance |
| `NOTIFICATION_SENDER` | — | Email address (shared mailbox) for TAP notification emails. If empty, no notifications are sent |

const msalConfig = {
  auth: {
    clientId: "<client-id>",
    authority: "https://login.microsoftonline.com/<tenant-id>",
    redirectUri: window.location.origin,
  },
  cache: {

  },

  scopes: ["User.Read", ...],

This file is generated automatically by the deploy script. Do not commit it to source control.

## Theming

Both portals use CSS Custom Properties defined in `css/theme.css`. This is the **single file** you need to edit for branding.

### Files to customise

| File | Purpose |
|------|--------|
| `logo.png` | Company logo – shown in the top bar (28 px height) and footer (16 px). Just replace the file. |
| `favicon.png` | Browser tab icon. Just replace the file. |
| `css/theme.css` | All colours, fonts, border radii, shadows – fully commented. |

### CSS Variable Reference (Light Mode)

```css
:root {
  /* ── Primary brand colour ─────────────────── */
  --color-primary:        #0078d4;       /* Main accent (buttons, links)    */
  --color-primary-hover:  #106ebe;       /* Button hover                    */
  --color-primary-active: #005a9e;       /* Button pressed / active         */
  --color-primary-light:  #deecf9;       /* Subtle highlight backgrounds    */
  --color-primary-text:   #ffffff;       /* Text on primary-coloured bg     */

  /* ── Semantic colours ─────────────────────── */
  --color-success:        #107c10;       /* Success text / icons            */
  --color-success-bg:     #dff6dd;       /* Success background              */
  --color-danger:         #a4262c;       /* Error / danger text             */
  --color-danger-bg:      #fde7e9;       /* Error background                */
  --color-warning:        #7a6c00;       /* Warning text                    */
  --color-warning-bg:     #fff4ce;       /* Warning background              */

  /* ── Surface & background ─────────────────── */
  --color-bg:             #f5f5f5;       /* Page background                 */
  --color-surface:        #ffffff;       /* Cards, panels, top bar          */
  --color-border:         #e8e8e8;       /* Borders & dividers              */

  /* ── Typography ───────────────────────────── */
  --color-text:           #202020;       /* Default body text               */
  --color-text-muted:     #717171;       /* Secondary / helper text         */
  --font:                 'Segoe UI', system-ui, -apple-system, sans-serif;
  --font-mono:            'Cascadia Code', 'Consolas', monospace;

  /* ── Layout & shape ──────────────────────── */
  --radius:               8px;           /* Default border-radius           */
  --radius-lg:            12px;          /* Cards & larger elements         */
  --topbar-h:             56px;          /* Top navigation height           */
}
```

### Dark Mode Overrides

In the `[data-theme="dark"]` block, only list the variables that differ from light mode. Everything else is inherited.

```css
[data-theme="dark"] {
  --color-bg:             #1a1a2e;
  --color-surface:        #22223a;
  --color-border:         #333352;
  --color-text:           #e0e0e0;
  --color-text-muted:     #9a9ab0;
  --color-primary-light:  #1a3a5c;
  --color-success-bg:     #1a3a1a;
  --color-danger-bg:      #3a1a1a;
  --color-warning-bg:     #3a3a1a;
}
```

## Table Storage Schema

### TapAuditSelfService / TapAuditHelpdesk

| Column | Type | Description |
|--------|------|-------------|
| `PartitionKey` | String | Date `YYYY-MM-DD` |
| `RowKey` | String | UUID v4 |
| `Portal` | String | `selfservice` or `helpdesk` |
| `AgentUPN` | String | Helpdesk agent UPN |
| `AgentId` | String | Helpdesk agent OID |
| `TargetUPN` | String | Target user UPN |
| `TargetId` | String | Target user OID |
| `Action` | String | `SUCCESS`, `DENIED`, `BLOCKED_PRIVILEGED`, `RATE_LIMITED`, `ERROR` |
| `DenialReason` | String | Reason for failure |
| `TapLifetimeMinutes` | Int32 | Configured TAP lifetime |
| `TapIsUsableOnce` | Boolean | One-time use flag |
| `ClientIp` | String | Client IP address |
| `UserAgent` | String | Browser user agent |

### RateLimitTracking

| Column | Type | Description |
|--------|------|-------------|
| `PartitionKey` | String | `agent-{id}` or `target-{id}` |
| `RowKey` | String | Timestamp-based unique key |
| `TargetUserId` / `AgentId` | String | Cross-reference ID |
| `ExpiresAt` | String (ISO 8601) | When the TAP expires |
